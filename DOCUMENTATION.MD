**Only for russian language.**

Параметры:
```
 -ag66u_noupdater - отключить апдейтер
 -ag66u_dev - режим разработчика AG 6.6-U
 -ag66u_hdtextures - поддержка 1024x1024/2048x1024 текстур для моделей
```

Команды и квары:
```
ag66u_crash - крашнуть клиент (работает только при -ag66u_dev)
ag66u_changelog - показать ченджлог последней версии
ag66u_checkupdate - проверить клиент на обновления
ag66u_version - показывает версию вашего клиента
ag66u_updateclient - обновить клиент
ag66u_ignore_update_checking - игнорировать проверку обновлений при обновлении клиента
ag66u_show_updatemsg - показывать уведомление о новом обновлении вверху
cl_autorecord - автоматическая запись демо при отсчёте начала матча
cl_autostop - автоматическая остановка записи демо после матча
cl_autowepswitch - автопереключение оружия при подборе
cl_autojump - автоматический распрыг при задевании поверхности
cl_bob_angled - покачивание из WON версии
cl_chat_log - логгировать чат в .log файл
cl_cross - включить кастомный прицел (1 - настраиваемый, 2 - прицел =Maverick=)
cl_cross_color - цвет прицела
cl_cross_alpha - прозрачность прицела
cl_cross_thickness - толщина прицела
cl_cross_size - размер прицела
cl_cross_gap - зазор прицела
cl_cross_outline - контур прицела
cl_cross_circle_radius - радиус круглого прицела
cl_cross_dot_size - размер точки прицела
cl_cross_dot_color - цвет точки прицела
cl_cross_top_line - верхняя линия прицела
cl_cross_bottom_line - нижняя линия прицела
cl_cross_left_line - левая линия прицела
cl_cross_right_line - правая линия прицела
cl_disable_deadcam - отключает поворот камеры при смерти
cl_disable_decaltraces - отключает следы от пуль
cl_disable_egon_beam - отключает лазер от егона
cl_disable_gauss_balls - отключает шарики, оставляемые после выстрела гауссом
cl_disable_gauss_beam - отключает лазер от гаусса
cl_dynamic_fov - динамическое изменение FOV'а при передвижении
cl_dynamic_fov_denom - знаменатель изменения FOV'а при передвижении
cl_forceenemymodel - заменяет модель врага на вашу
cl_forceteammatemodel - заменяет модель тиммейта на вашу
cl_forceteammodel - заменяет модель определенной команды на вашу
cl_forceteammodel_remove - убирает для определенной команды вашу модель
cl_forceteammodel_list - список команд, где задействована замена на вашу модель
cl_gaussbeam_color - цвет луча гаусса ("R G B")
cl_retract_weapon - при приближении к стенке или другому загораживающему объекту, модель оружия отрекается от стенки
cl_killsound - звук при убийстве врага
cl_killsound_path - путь к звуку при убийстве врага
cl_messages_log - логгировать пакеты, посылаемые сервером
cl_messages_dump - сдампить лог пакетов
cl_hidecorpses - скрывает забагованные тела мёртвых игроков (РЕКОМЕНДУЕТСЯ)
cl_hl2_bob - покачивание из Half-Life 2
cl_hl2_weaponlag - задержка поворота оружия из Half-Life 2
cl_protect - помощь по защите от слоухакинга
cl_protect_log - лог того, что заблокировал защитник
cl_protect_block - заблокировать исполнение команды со стороны сервера (например, retry)
cl_protect_allow - разрешить команду
cl_protect_block_cvar - заблокировать квар (например, rate)
cl_righthand - при значении 1 меняет положение модельки оружия влево и наоборот (уж лучше бы я переименовал в cl_lefthand)
cl_shadows - тени
cl_shadows_entities - тени от энтитей
cl_shadows_players - тени от игроков
cl_showfps_pos - позиция FPS строки
cl_showfps_color - цвет FPS строки (RGB)
cl_showinterstats - показывает статистику использования оружия в конце матча
cl_viewmodel_fov - отдельный угол обзора для модели оружия
cl_viewmodel_hltv - возможность сразу в игре сделать себе HLTV модельки оружий, что отключает анимации бездействия, доставания и т.д
cl_viewmodel_semitransparent - делает модели оружия полупрозрачными
cl_votesound - звук при голосовании
customtimer_precision - количество цифр после запятой для customtimer команды
dl_realnames - скачать реалнеймы. Берёт ссылку с dl_realnames_url
dl_realnames_url - ссылка на скачивание реалнеймов
hud_alwaysactive - делает весь HUD активным (прозрачность ставится на 255)
hud_breathstatus - статус дыхания под водой
hud_customsprite - включает ваш кастомный спрайт
hud_customsprite_name - путь до вашего спрайта
hud_customsprite_x - координата X вашего спрайта (ширина)
hud_customsprite_y - координата Y вашего спрайта (высота)
hud_customsprite_mode - режим рисования вашего спрайта (0 - обычный, 1 - прозрачный, 2 - аддитивный)
hud_speedometer - показывает вашу скорость (полезно в KreedZ)
hud_speedometer_type - тип спидометра (0 - обычный, 1 - Maverick style)
hud_jumpspeed - показывает скорость при прыжке (полезно в KreedZ)
hud_splash - позволяет включить/выключить сплэш fisk'а в режиме спектатора
hud_battery_width_pos - горизонтальная позиция брони
hud_battery_top - поставить броню поверх здоровья
hud_battery_hide - скрывает броню при значении нуль
hud_colortext - цветной вывод некоторых сообщений в консоли
hud_gaussstatus - статус зарядки гаусса
hud_health_wasted - элемент HUD'а, показывающий изменение HP
hud_nametags - показывает имя над головой ваших тиммейтов
hud_nametags_avatar - показывает аватары из Steam над головой ваших тиммейтов
hud_nametags_avatar_outline_color - указывает цвет обводки аватаров (RGB)
hud_origin - ваши координаты
hud_reloadstatus - статус перезарядки оружия
hud_reloadstatus_pos - позиция статуса перезарядки оружия
hud_scoreboard_color - цвет обычного скорборда (R/G/B/ИНВЕРТ-A)
hud_scoreboard_kd - графа К/Д в обычном скорборде
hud_weapon_denom_x - знаменатель для X координаты (ШИРИНА ЭКРАНА делится на значение hud_weapon_denom_x)
hud_weapon_denom_y - знаменатель для Y координаты (ОФФСЕТ СПРАЙТА ОРУЖИЯ делится на значение hud_weapon_denom_y)
hw_disable_dynamic_lightmaps - отключает абсолютно весь динамический свет в игре (лучшая версия r_dynamic_ent_light)
hw_disable_mdl_effects - отключает эффекты от моделек некоторых
hw_disable_mdl_textures - отключает текстуры моделей и заменяет их на последнюю загруженную
hw_disable_beamfollow - отключает след от RPG ракеты
hw_disable_sparks - отключает искры, появляющиеся при взрывах
hw_fix_hud_blur - фиксит оригинальный баг движка, который мылит спрайты из-за неправильного масштабирования
hw_skybox_color - цвет скайбокса ("R G B A"). Работает только при hw_skybox_disable
hw_skybox_disable - отключить скайбокс
hw_load_skybox - загрузить скайбокс с необходимым названием (например, "desert")
joinlast - команда позволяет вам заходить на последний сервер, посещаемый вами. Походит на retry, но работает даже после перезахода в игру (фича Rofi's Client DLL)
r_refreshrate - отвечает за частоту обновления, редко нужна.
_zpecial - позволяет зацикливать алиасы
+duckroll - спам приседания
+ducktap - это техника, позволяющая делать подобие баннихопа без +jump, но с +duck. С помощью Дактапа можно выиграть немного времени в воздухе, чтобы выполнить какой-либо бхоп паттерн.
+ljump - сделать длинный прыжок одной кнопкой
r_dynamic_ent_light - при значении 0 убирает динамический свет со всех энтитей (применимо к RPG ракете)
r_playeranims - при значении 0 убирает анимации игрока
r_playeranims_gait - при значении 0 убирает gaitsequences у игрока
snapshot_original - делать скрины в оригинальном формате .bmp, игнорируя все свистоперделки AG 6.6-U
snapshot_folder - сохранять PNG скрины в отдельную папку screenshots (не работает с snapshot_original)
snapshot - сделать BMP или PNG скриншот в папку ag или ag/screenshots взависимости от конфигурации двух предыдущих команд.
steam_notifypos - позиция Steam уведомлений (требует перезахода в игру)
togglefullscreen - переключиться на оконный режим налету (fs - сокращение этой команде, на высоких разрешениях, пересекающихся с виндовсовским разрешением, могут быть артефакты)
```
Все остальные команды от стандартной HL и AG 6.6.
